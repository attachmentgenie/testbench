Testbench is a small PHP test framework to investigate slow database queries.

## Install

Via Composer

``` bash
$ composer require attachmentgenie/testbench
```

## Usage

``` php
./vendor/bin/testbench --help
```

## Testing

``` bash
$ ./bin/phpunit -c phpunit.xml.dist
```
